Custom Fields

Share

Overview

Custom field is a field defined by administrator. Custom field is used to store any additional information.

  • Custom field can be associated with one, or more workflows.
  • Each workflow may have different custom fields. It is possible to define in which actions will a custom field be available.
  • Custom fields can be added to any action on the pages for managing workflow. Later, using that action, users can view, delete or modify custom fields.
  • There are default custom fields, called standard fields which are embedded in all workflows by default.
  • Custom field can be added to Contacts, Accounts, Users or Projects.
  • Custom fields added to contacts, accounts or projects are visible on the user side, on pages for viewing them. In administration, it is possible to set who will be able to view and edit those custom fields.
